En esta guía se explica cómo bastionar el arranque de un sistema Debian, asegurando el control sobre el cargador de arranque (GRUB), activando mecanismos de seguridad como Secure Boot y TPM, y reduciendo el riesgo de arranques no autorizados o manipulaciones del sistema operativo.

Antes de comenzar, asegúrate de tener tu sistema Debian actualizado y sin paquetes corruptos.

sudo apt update && sudo apt upgrade -y
sudo apt install debsums -y
sudo debsums -s

El cargador de arranque GRUB puede ser manipulado si no se protege adecuadamente. Para evitar accesos no autorizados, establece una contraseña para el usuario administrador del GRUB.

  1. Genera una contraseña cifrada:
grub-mkpasswd-pbkdf2

Copia la salida que comienza con grub.pbkdf2.sha512.10000...

  1. Edita el archivo de configuración personalizada de GRUB:
sudo nano /etc/grub.d/40_custom
  1. Añade estas líneas (reemplaza la clave cifrada con la tuya):
set superusers="admin"
password_pbkdf2 admin grub.pbkdf2.sha512.10000.xxxxxxxxxxxxxxxxxxxxxxxxx
  1. Guarda los cambios y actualiza el cargador:
sudo update-grub

Ahora, el acceso al menú de arranque GRUB requerirá autenticación.

Una vez configurada la contraseña, actualiza GRUB para aplicar los cambios:

sudo update-grub

Este comando regenerará la configuración del cargador de arranque y aplicará la protección por contraseña.

Reinicia el sistema y, en el menú de arranque, pulsa e sobre la entrada de Debian.

Debería solicitarte la contraseña configurada para poder editar los parámetros de arranque. Esto confirma que la protección está correctamente activada.
